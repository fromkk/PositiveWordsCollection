# ポジコレ-PositiveWordsCollection-
ポジティブな言葉をつぶやくSNSです。
嬉しかったことを投稿して共有したり、コメント機能で前向きな言葉を掛け合い、応援し合うことができます。
嬉しかった言葉を日記につけるようにつぶやくことで、ハッピーな気持ちをいつでも見返すことができます。
